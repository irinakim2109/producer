# Контент-агент 🎬

Это дом твоего личного **агента-продюсера** для курса «Вайб на миллион». Внутри - готовый рабочий стол, память и инструменты, чтобы агент помогал тебе делать контент: идеи, сценарии, ролики, посты, воронку.

Ты разворачиваешь этот дом один раз, а дальше просто разговариваешь с агентом словами - как с живым помощником. Терминала бояться не надо.

---

## Что где лежит

Репозиторий состоит из двух частей:

- **Дом агента** (всё в корне: `CLAUDE.md`, `USER.md`, `knowledge/`, `.claude/skills/` и т.д.) - его ты разворачиваешь себе на компьютер.
- **Склад скиллов** (папка `skills/`) - магазин инструментов на будущее. Его ты НЕ забираешь целиком: агент докупает оттуда по одному скиллу, когда на курсе доходишь до нужной недели.

### Внутри дома

| Папка / файл | Что это простыми словами |
|---|---|
| `CLAUDE.md` | Мозг агента - кто он и как работает (не трогай) |
| `SOUL.md` | Характер агента - каким тоном говорит |
| `USER.md` | Твоя анкета - кто ты, про что твой контент (заполнишь на распаковке) |
| `MEMORY.md` | Что агент помнит про тебя всегда |
| `memory/` | Дневник - что делали по дням |
| `knowledge/` | Копилка знаний: твоя аудитория, темы, идеи |
| `content/` | Твои готовые работы: идеи, сценарии, посты |
| `.claude/skills/` | Включённые инструменты агента (на старте их три) |
| `skills/` | Склад: инструменты на будущее, докупаются по ходу курса |

## Как поставить дом (одна команда агенту)

Открой Claude Code в пустой папке и попроси агента человеческими словами:

> Склонируй репозиторий https://github.com/Ntmib/content-agent в эту папку, но НЕ скачивай папку `skills` (это склад на будущее). Потом отвяжи папку от git и расскажи простыми словами, что за папки появились.

Агент сделает всё сам. Если ему нужны точные команды - вот они:

```bash
git clone --no-checkout --depth 1 https://github.com/Ntmib/content-agent.git .
git sparse-checkout init --no-cone
printf '/*\n!/skills/\n' > .git/info/sparse-checkout
git checkout main
rm -rf .git   # отвязать от общего репо - дом теперь твой
```

После этого напиши агенту `привет` - он представится и покажет меню. Значит дом работает.

---

## Как докупать скиллы

Каждую неделю курса открывается новый инструмент. Когда на уроке скажут - просто попроси агента словами:

> поставь скилл сценарист

Сработает робот-установщик: возьмёт нужный скилл со склада `skills/`, включит его агенту и объяснит, как проверить. Потом начни новый чат - и агент уже умеет новое.

Почему не всё сразу? Чтобы не заваливать тебя инструментами в первый день. Каждый приходит тогда, когда он на курсе реально нужен.
