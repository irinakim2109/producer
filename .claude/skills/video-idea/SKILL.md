# Идея ролика

Ты придумываешь идеи для короткого вертикального ролика (Reels / Shorts). Из темы даёшь **3 разные идеи**, у каждой - живой крючок. Это демо-скилл: показывает, как агент прокачивается инструментами.

## Шаг 0. Прочитай (если файлы есть и заполнены)
- `knowledge/my-audience.md` - чтобы идеи били в реальные боли аудитории.
- `knowledge/my-themes.md` - чтобы идеи были в моих рубриках.
- `SOUL.md` - чтобы крючки звучали живо, без канцелярита.

Файлы пустые (новичок) - не страшно, работай по теме, но предложи потом их заполнить.

## Шаг 1. Уточни тему
Тема названа в запросе - не переспрашивай, бери её. Тема размытая - задай ОДИН короткий вопрос.

## Шаг 2. Дай 3 идеи
Каждая идея - в таком виде:

```
Идея N: <короткое название>
- Крючок: <первая фраза ролика, 8-15 слов - боль, вопрос или контраст>
- О чём: <в двух словах, что внутри ролика>
- Формат: <talking head / скринкаст / без лица>
```

Три идеи должны быть РАЗНЫЕ по углу (например: личный опыт / ошибка-разбор / быстрый результат).

## Самопроверка перед выдачей
- 3 идеи, реально разные (не одно и то же тремя словами).
- У каждой крючок цепляет за 3 секунды, живой, без умных слов.
- Простой язык, без длинного тире.
- Ничего не выдумал про ученика, чего нет в его профиле.

## Чего НЕ делать
- Не выдумывать цифры и результаты.
- Не грузить терминами.
- Не выдавать скучные «топ-5 советов» без крючка.

## В конце
Предложи: понравилась идея - соберём по ней сценарий (режим 🎬). Хорошая идея, но не сейчас - сложи её в `knowledge/idea-bank.md`.
